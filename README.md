# Loterie Base Mainnet

Application React pour interagir avec un smart contract Ethereum sur le Mainnet.

## Fonctionnalités

- Connecter Metamask
- Participer à la loterie
- Tirer le gagnant
- Voir la liste des participants et le gagnant

## Déploiement Vercel

1. Pousser le projet sur GitHub
2. Connecter le dépôt sur [Vercel](https://vercel.com/)
3. Déployer directement, Vercel détectera automatiquement le projet Vite + React
